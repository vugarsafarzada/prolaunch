App development zamanı biz əsasən terminaldan istifadə edirik. Məsələn, frontend yada backend ilə çalışan hər insan öncə terminalı açır, kommand ilə run edir. 

İdea odur ki bir app quraq hansı ki mac, windows, linux hamısında işləsin. Öz interfeysi olsun. Misal üçün bütün JavaScript/TypeScript projectlərində run üçün lazım olan commandların hamısı package.json faylında yazılır. Yaratdığımız app hansı folderdə açılarsa ordakı package.json faylını oxuyur və kommandaları çıxarıb userə interface olaraq (Buttonlar şəklində göstərir). Və user o buttonlara click edən zaman arxada o command çağrılır və loglar userə göstərilir. Əgər kill etmək istəyərsə də ordan kill edir. Hansı portda hansı proses işlədiyini görür, idarə edir. Yəni terminalda yazılan kommandaları biz bir interfeysə yığırıq. Beləliklə devlərin işi çox rahatlaşır, əziyyət çəkmirlər terminal qarmaşığından. 

App öz daxilində ilk açılanda proyekt yarat yada mövcud olanı aç menusu ilə gəlir. Əgər yenisini yarat seçilərsə seçimlər gəlir məsələn next js next react seçə bilirsən yanında vite ilə də yarada bilirsən və sairə. Sonra user özünə qovluq seçir və proyekt öz özünə quraşdırılır və həmin folder də açılır kommandlar və loglar görünür. 

Yəni Create project vurandakı haldı flow belə davam edir
1. Create projectə vurulur
2. Gallery açılır. Galleri daxilində tablar və o tablar proqramlaşdırma dilləridi. indiki case də sadəcə 3 tab var All, JavaScript və TypeScript
3. User tablarla filter edir yada search ilə all-da axtarır.
4. Burada son versiya create react app və ya vite yarada bilər. Məsələn NextJS-in son versiyası v16-dir eyni zamanda qədim v14 və ya v15 seçib yarada bilər. Eyni qayda ile digerleri üçün də. Eyni ilə Vue, Svelte və Angular üçün də eyni ilə.
5. Select edildikdən sonra NEXT düyməsi aktiv olur. Bu NEXT düyməsinə vurulanda folder seçmək üçün yer açılır və name verilir. Sonra create düyməsi aktiv olur. Create vurulanda lazımlı kommandlar ilə create edilir və open exist project avtomatik işə düşüb həmin yer açılır və həmin layihənin run kommandları button olaraq çıxır və logları görünür.

Əgər mövcud olanı aç seçilərsə mövcud folder seçib açır və o folderə aid olan commandları interfeysə yerləşdirir buttonlar və loglar şəklində. Bunlardan əlavə tablar da var + iconuna vurub eyni pəncərədə bir neçə iş görə biləcək.
Zamanla tək JS/TS deyildiginən başqa dillərdə də dəstəkləyəcək bunu nəzərə alaq ki sonra da problem olmasın, amma indi ilk iş JavaScript/TypeScript ailəsi üçün focus edək. Əsas idea terminalsız sadəcə bu desktop app ilə istifadəçini visual olaraq təmin etməkdi. 

İstifadəçilər bu app -ə windows, macos, linux sistemlərində rahatlıqla yükləyib istifadə edə bilsinlər. Yükləmək üçün isə 1 yol var npm pnpm və yarn vasitəsi ilə.
